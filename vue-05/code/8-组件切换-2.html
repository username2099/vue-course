组件切换 VUE 官网

Login   Register





要求：

1. 页面刷新只显示登录

2. 点击 Login 只显示登录 且界面不能刷新

3. 点击 Register 显示注册 且界面不能刷新

4. 利用之前所学的知识，比如 if else show 指令

5. 点击 Register ，Login通过动画移出，Register 通过动画移入

6. 点击 Login ，Register通过动画移出，Login 通过动画移入

附加[不做要求]：界面刷新，Login通过动画移入，组件切换时先移出后移入，使用css动画库或js库和钩子函数
